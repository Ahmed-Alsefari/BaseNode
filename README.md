# BaseNode
A lightweight, free personal server for remote file access from any browser, keeping your files private and local.



The Problem: 

• No simple, free way to access your home PC files remotely. 

• Options are either costly (cloud storage), overkill (remote desktop), or only work on your home network. 




My Solution : 

• A lightweight personal server you install on your PC. 

• Access files from any browser anywhere (phone/laptop/tablet). 

• Your files stay on your computer (private & free) . 

• Simple interface: just upload, download, and browse. 

• Secure login to protect access . 

• Designed for one user - no complicated setup needed.
